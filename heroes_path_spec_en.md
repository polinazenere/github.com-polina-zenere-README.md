# Goals and Objectives

## Main Goals
- Create an engaging platformer with moral choices
- Implement a reputation system with visible consequences
- Develop diverse superpowers

## Technical Tasks
1. Implement basic platformer mechanics
2. Create a reputation system with three factions
3. Develop a skill tree
4. Set up a progress saving system

# Target Audience
- **Age:** 12+
- **Platform:** iOS/Android
- **Interests:** Platformers, comics, games with moral choices

# Genre and Key Features
## Genre
2D platformer with RPG elements and moral choice

## Key Features
- Platformer - precise controls, challenging jumps
- Superheroes - unique abilities and costumes
- Reputation System - choices with long-term consequences

# Technical Requirements
## Minimum Requirements
- iOS: iOS 14+
- Android: Android 9.0+

## Target Specifications
- Performance: stable 60 FPS
- Loading: < 2 seconds
- Size: < 500 MB

# Platformer Content
## Level Types
1. City Rooftops - classic platforming
2. Dungeons - challenging jumping sections
3. Skyscrapers - vertical levels
4. Secret Bases - stealth elements

# Reputation System
## Reputation Scales
- **Guardians of Law** (0-100) - order and discipline
- **Defenders of Freedom** (0-100) - freedom and rights
- **Shadow League** (0-100) - pragmatism and efficiency

## Consequences
- Access to abilities
- NPC reactions
- Hero appearance
- Different endings

# Skill Tree
## Strength (Guardians of Law)
├─ Super Strike
├─ Defense

## Speed (Defenders of Freedom)
├─ Quick Dash

## Stealth (Shadow League)
